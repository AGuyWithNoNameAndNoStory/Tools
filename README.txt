This program is used to create a .cue file in specified folder.
You should put cueCreater.exe in the folder of audio files where you want to create the playlist.
Run the cueCreater.exe and enter the content as required.
Then, you can get a .cue file.
You can select any audios whatever you want when you run the .cue file.
This program supports to extract .ape, .mp3 and.flac audio files.
